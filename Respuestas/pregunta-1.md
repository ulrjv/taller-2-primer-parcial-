## Tipo de base de datos

El sistema descrito, que busca documentos por **similitud semántica usando vectores**, corresponde a una **Base de datos vectorial** (*Vector Database*).

Este tipo de base de datos almacena datos como vectores numéricos de alta dimensión (embeddings) y permite realizar búsquedas por similitud semántica usando métricas como la **distancia coseno** o la **distancia euclidiana**, en lugar de búsquedas por coincidencia exacta.

**Ejemplos:** Pinecone, Weaviate, Qdrant, Milvus, y también MongoDB Atlas Vector Search.
